多位随机数可过期Token

发表于 2019-06-23 | 分类于 技术
ReadMe：

51token总算开张了，目前只做了可过期token，功能会越来越多的哦。



接口：Token生成
链接例子：

http://www.51token.xyz/get4DigitToken?schema=&length=&expire=

http://www.51token.xyz/get4DigitToken?schema=51token&length=9&expire=60

接口描述：
参数名 说明 类型 默认值 检查 备注
schema 区分因子 String 51token 可以不填
可以是任意字符
不要超过50位
输入检查不通过
则变成默认值
length 位数 Integer 4 可以不填
只能填1-9当中的一个数
输入检查不通过
则变成默认值		 输入4则返回4位Token
输入9则返回9位Token
expire 过期时间 Integer 30 可以不填
必须大于0
小于60*60*24
输入检查不通过
则变成默认值
返回内容：


接口：Token验证
链接例子：

http://www.51token.xyz/verify4DigitToken?schema=51token&token=244121747

接口描述：
参数名 说明 类型 默认值 检查 备注
schema 区分因子 String 51token 可以不填
可以是任意字符
不要超过50位
输入检查不通过
则变成默认值
token token String token输入不正确
或者时间过期
则校验失败		 之前生成的Token
返回内容：

安装

进入https://www.uipath.com.cn/点击“开始试用”

选社区版

随便填一下点提交

会收到一封邮件

就可以下载一个UiPathStudioSetup.exe的文件，安装完之后长这样

如果你以上操作都懒得的做，可以直接下我百度网盘上的文件：

官方教程

内容

我自己想做的内容还挺多：

  • 比如我想把SpringCloud，Docker，K8S，ELK一系列的东西都做一遍
  • 然后我还想做一个叫开源开箱的视频，现在开源软件，不都号称开箱即用吗，再加上开箱的视频又比较火，那开源开箱肯定会火吧。而且开源软件这么多，做个10年肯定没问题。我特别想做deepfake这种人工智能，图形处理的换脸开源开箱，因为这种特别好玩。
