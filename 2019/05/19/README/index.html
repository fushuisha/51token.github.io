ReadMe

各位小伙伴你们，这是我的博客，主要存放帮助文档和主播的技术博客
我之前做过一些视频博客，放到B站了
后面我做视频博客，都会在这里发布一个对应的MD，算是相互呼应，相互导流吧

内容

我自己想做的内容还挺多：
  • 比如我想把SpringCloud，Docker，K8S，ELK一系列的东西都做一遍
  • 然后我还想做一个叫开源开箱的视频，现在开源软件，不都号称开箱即用吗，再加上开箱的视频又比较火，那开源开箱肯定会火吧。而且开源软件这么多，做个10年肯定没问题。我特别想做deepfake这种人工智能，图形处理的换脸开源开箱，因为这种特别好玩。
  • 我又想做游戏视频。比如皇室战争，怪猎世界，这两款现在玩的比较多的游戏，我已经在B站发过几十个视频了，所以还是想继续做下去。
  • 域名叫做51token，主要考虑到token这东西，现在用的特别多。比如验证码就是一种token。为了解决CSRF问题也可以使用token。Oauth2.0中有了JWT（JsonWebToken）。那么能不能把各种token做成一种公共服务，微服务，或者是小程序呢。

总结

总之，我先把博客建起来，很多想做的事情，再慢慢丰富完善吧。
